# RC4
CS4236 Cryptography Theory and Practices Project

Implemented Attacks:
1) FMS Attack
2) Korek Attacks
  a) A_s5_1
  b) A_s13
  c) A_u13_1 
  d) A_u5_1 
  e) A_u5_2 
  f) A_u13_2 
  g) A_u13_3 
  h) A_u5_3 
3) Collective Attack -> FMS and korek attacks in a loop.
4) Selective Exhaustive Search -> for last 3 key bytes.
